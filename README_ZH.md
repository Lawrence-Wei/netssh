# Netssh

[English version](README.md)

![Netssh 应用截图](docs/assets/netssh-app-screenshot.png)

**Netssh** 是一款 Windows 桌面 SSH 与串口控制台工作站，面向运维、网络工程师、SRE 和 IT 管理员，帮助您在本地管理资产并快速连接设备。

## 安装

当前版本：**v1.1.9**

从 [GitHub Releases](https://github.com/team-gabage/netssh/releases) 下载安装包：

- **NSIS 安装包**（推荐）— 适合大多数 Windows 用户
- **MSI 安装包** — 适用于企业批量部署

## 快速开始

### 1. 添加主机

打开 Netssh 后，点击左侧面板底部的 **"+ New host"** 填写连接信息：

| 字段 | 说明 |
|------|------|
| 别名 | 主机的显示名称，如 "核心交换机" |
| 主机名 | IP 地址或域名，如 `192.168.1.1` |
| 用户 | SSH 登录用户名，如 `root` |
| 端口 | SSH 端口，默认 `22` |
| 站点 | 所属站点（如上海、云端），方便分组管理 |
| 备注 | 任意补充信息 |

点击 **Save** 保存。支持两种连接方式：

- **SSH** — 常规远程连接
- **串口** — 通过 COM 端口连接交换机/路由器控制台

### 2. 连接到主机

- **单击**左侧列表中的主机 → 右侧显示详情面板
- **双击**主机 → 立即打开 SSH 连接
- **右键**主机 → 可选"连接"、"编辑"、"收藏"、"删除"等操作

首次连接某台主机时，会弹出主机密钥确认窗口，验证指纹后即可信任。

### 3. 从 SSH Config 导入

点击左下角 **"Import config"**，选择 **"Read ~/.ssh"** 即可从 `~/.ssh/config` 批量导入已有主机。也支持导入 Excel、JSON、CSV 文件。

导入前会展示预览与诊断结果（重复别名、缺失密钥等），确认后才写入。

### 4. 手动连接（一次性）

不想保存到主机列表？在详情面板中使用手动连接功能，输入 IP/域名、端口、用户名和密码即可打开一次性 SSH 会话。

## 主要功能

### 主机管理
- **站点分组** — 按地理位置或用途将主机归类（上海、云端、实验室等）
- **标签与收藏** — 标记常用主机，快速筛选
- **搜索过滤** — 搜索别名、IP、标签；按"全部 / 收藏 / 最近"过滤
- **SSH 配置导入** — 从 `~/.ssh/config` 只读导入，不会修改原文件

### 终端会话
- **多标签页** — 同时打开多个 SSH / 本地 Shell 标签页
- **四象限分屏** — 将终端区域拆分为最多 4 个面板
- **本地 Shell** — 通过 Windows ConPTY 内嵌 PowerShell、CMD 或 WSL
- **命令片段** — 预置常用命令（系统、网络、Docker、监控等），拖拽或双击执行
- **连接状态栏** — 显示延迟、加密算法、会话时长等信息

### 安全
- **凭据保护** — 密码和密钥口令存储在 Windows 凭据管理器 / 密钥环中，不会写入本地文件
- **主机密钥验证** — 首次连接须确认 TOFU（首次信任），密钥变更时阻断连接
- **日志不记录命令** — 操作日志不含用户命令文本

### 外观与偏好
- **主题** — 极光紫、钴蓝、Windows Mica 三种主题
- **语言** — 中英文界面切换，终端 LC_ALL/LANG 环境变量跟随
- **毛玻璃效果** — 跟随 Windows Acrylic 半透明效果
- **字体与光标** — 终端字体、大小、光标样式自定义
